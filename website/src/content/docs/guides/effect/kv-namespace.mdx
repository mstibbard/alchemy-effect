KV is Cloudflare's global, low-latency key-value store. With Effect
bindings, you get a typed wrapper with `get`, `put`, `list`, and
`delete` operations.

:::note
For the full API reference, see the [KVNamespace provider](/providers/cloudflare/kvnamespace).
:::

<Steps>

1. **Create a Worker with a KV Namespace**

   Create the namespace, bind it in the init phase, and use it in
   the `fetch` handler.

   ```typescript
   // src/Api.ts
   import * as Cloudflare from "alchemy/Cloudflare";
   import * as Effect from "effect/Effect";
   import { HttpServerRequest } from "effect/unstable/http/HttpServerRequest";
   import * as HttpServerResponse from "effect/unstable/http/HttpServerResponse";

   const MyKV = Cloudflare.KVNamespace("MyKV");

   export default class Api extends Cloudflare.Worker<Api>()(
     "Api",
     { main: import.meta.path },
     Effect.gen(function* () {
       const kv = yield* Cloudflare.KVNamespace.bind(MyKV);

       return {
         fetch: Effect.gen(function* () {
           const request = yield* HttpServerRequest;
           const url = new URL(request.url);
           const key = url.pathname.slice(1);

           if (request.method === "PUT") {
             const body = yield* request.text;
             yield* kv.put(key, body);
             return HttpServerResponse.text("Stored", { status: 201 });
           }

           const value = yield* kv.get(key);
           return value
             ? HttpServerResponse.text(value)
             : HttpServerResponse.text("Not found", { status: 404 });
         }),
       };
     }),
   ) {}
   ```

2. **Deploy**

   ```sh
   alchemy deploy
   ```
