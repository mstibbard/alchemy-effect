This guide walks you through creating and deploying a Cloudflare Worker
with alchemy-effect.

:::note
For the full API reference, see the [Worker provider](/providers/cloudflare/worker).
For async, inline, and modular patterns, see the [Platform concept](/concepts/platform).
:::

<Steps>

1. **Define the Worker class**

   Create a file for your Worker. The `main` prop tells Alchemy which
   file to bundle and deploy.

   ```typescript
   // src/Api.ts
   import * as Cloudflare from "alchemy-effect/Cloudflare";
   import * as Effect from "effect/Effect";
   import * as HttpServerResponse from "effect/unstable/http/HttpServerResponse";

   export default class Api extends Cloudflare.Worker<Api>()(
     "Api",
     { main: import.meta.path },
     Effect.gen(function* () {
       return {
         fetch: Effect.succeed(HttpServerResponse.text("Hello")),
       };
     }),
   ) {}
   ```

2. **Add resource bindings**

   Bind KV, R2, Durable Objects, or other services in the init phase.
   They become available to all runtime handlers.

   ```typescript
   Effect.gen(function* () {
     // init — bind resources
     const kv = yield* Cloudflare.KVNamespace.bind(MyKV);
     const counters = yield* Counter;

     return {
       // runtime — use them
       fetch: Effect.gen(function* () {
         const value = yield* kv.get("key");
         const counter = counters.getByName("default");
         yield* counter.increment();
         return HttpServerResponse.text(value ?? "no value");
       }),
     };
   })
   ```

3. **Create the Stack**

   Wire the Worker into your `alchemy.run.ts` stack file.

   ```typescript
   // alchemy.run.ts
   import * as Alchemy from "alchemy-effect";
   import * as Cloudflare from "alchemy-effect/Cloudflare";
   import * as Effect from "effect/Effect";
   import Api from "./src/Api.ts";

   export default Alchemy.Stack(
     "MyApp",
     { providers: Cloudflare.providers() },
     Effect.gen(function* () {
       const api = yield* Api;
       return { url: api.url };
     }),
   );
   ```

4. **Deploy**

   ```sh
   alchemy deploy
   ```
